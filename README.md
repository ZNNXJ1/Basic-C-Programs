# 💻 Práticas de Lógica e Estrutura em C

Este repositório centraliza meus estudos em linguagem C, com foco em resolver desafios de lógica de programação de forma nativa e resiliente.

### 🧠 Filosofia de Desenvolvimento:
- **C Nativo:** Usando as funções padrão da linguagem (`stdio.h`, `string.h`, `ctype.h`).
- **Blindagem de Código:** Todo programa é desenvolvido com loops de validação (`do-while`) e tratamento de buffer, visando impedir que entradas inesperadas do usuário causem bugs ou comportamentos indefinidos.
- **Eficiência e Memória:** Atenção rigorosa aos tipos de dados (como `long long` para grandes volumes numéricos) para evitar problemas de transbordamento (*overflow*).

*Este é um repositório de evolução contínua, focado em códigos simples mas trabalhosos*
